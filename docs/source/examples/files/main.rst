
Αρχική υποβολή στοιχείων αναφοράς και λογιστικών στοιχείων
==========================================================

Στο παράδειγμα που ακολουθεί συμπληρώνονται για πρώτη (και για τελευταία εκτός
αν υπάρχουν αλλαγές ή προσθήκες) φορά όλα τα στοιχεία αναφοράς καθώς και τα
λογιστικά στοιχεία.  

Αφού υποβληθούν τα στοιχεία ενημερώνεται η IFDAT-LIST για επιλεγμένα στοιχεία
αναφοράς τόσο σε στατική μορφή (IFDAT_SNAPSHOT_REPORT) αλλά και σε δυναμική (IFDATFULL_REPORT) όπου
φαίνεται η εξέλιξη των μεταβλητών χαρακτηριστικών στη διάρκεια του χρόνου.
