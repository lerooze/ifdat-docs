
Πώληση μετοχής short
====================

Στην περίοδο Τ0 ο ΕΟ έχει 1000 ευρώ σε καταθέσεις:

===========  ========================

-------------------------------------
Μεταβλητή    Τιμή   
===========  ========================
STCK          1000        
FLS           0       
===========  ========================
 
===========  ========================
Τ0 ACC.KEY 
-------------------------------------
Μεταβλητή    Τιμή   
===========  ========================
BK_PRC        1000        
INCM          0       
===========  ========================
